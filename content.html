The id() Function

The id() function returns the memory address (identity) of an object:

x = [1, 2, 3] print(id(x))

The exact number depends on your computer and Python session, but it uniquely identifies that object in memory.

The hex() Function

Memory addresses are often displayed in hexadecimal (base 16). Use the hex function to convert to this format:

x = [1, 2, 3] print(hex(id(x)))
